# ML Lab Project - 6th Semester

Dataset: Parkinson's dataset (parkinsons.csv) used for classification and analysis exercises.

## Lab 1 - Intro to Data Analysis
Exploratory data analysis, basic statistics, and initial data inspection.

## Lab 2 - Data Preprocessing and Visualization
Data cleaning, handling missing values, feature scaling, and visualization techniques.

## Lab 3 - Descriptive Statistical Analysis
Summarizing datasets using measures of central tendency and dispersion.

## Lab 4 - Inferential Statistical Techniques
Hypothesis testing, confidence intervals, and drawing conclusions from samples.

## Lab 5 - Logistic Regression (housing)
Binary classification using logistic regression applied to housing-related features.

## Lab 6 - Support Vector Machine Classification
Support Vector Machines for classification, kernel selection, and model tuning.

## Lab 7 - Decision Tree Classification
Decision tree models, splitting criteria, pruning, and model evaluation.
